# Site

Набор статических HTML-страниц. Публикуется через **GitHub Pages**.

## Страница ЧМ-2026 — `wc2026.html`

Адаптивная страница чемпионата мира 2026: группы, сетка плей-офф и прогноз.
Открывается по адресу `https://<ваш-логин>.github.io/<репозиторий>/wc2026.html`.

### Что внутри
- **Мобильная вёрстка** — карточки групп в одну колонку на телефоне, тянущиеся вбок
  табы и сетка плей-офф со свайпом, учёт «чёлки» (safe-area), тап-таргеты от 44px.
- **Живые результаты** — таблицы групп подгружаются из бесплатного API
  [football-data.org](https://www.football-data.org/) прямо в браузере.
- **Прогноз** — сетка плей-офф, финал и блок «кто победит» считаются прогнозом и
  хранятся в самом файле (никакой API их не отдаёт).

### Включить живые данные
1. Зарегистрируйтесь: https://www.football-data.org/client/register
2. Скопируйте токен.
3. В `wc2026.html` найдите блок `const LIVE = {` и вставьте токен в `apiKey`.
4. Закоммитьте и запушьте.

Строка статуса над таблицами показывает режим: 🟢 «в реальном времени» или
⚪ «снимок данных». Без ключа или при недоступном API страница работает на
встроенном снимке — ничего не ломается.

> Ключ на публичном сайте виден всем — для бесплатного тарифа football-data.org
> это допустимо (лимит запросов, без платных данных). Не используйте здесь
> платные ключи.
>
> Если браузер блокирует прямой запрос (CORS), пропишите CORS-прокси в поле
> `corsProxy` в том же блоке `LIVE`.

## Публикация (GitHub Pages)
Settings → Pages → Source: ветка `main`, папка `/ (root)`. Файл `.nojekyll`
уже в репозитории — он отключает обработку Jekyll и отдаёт файлы как есть.
