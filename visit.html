  • Отсутствие синхронизации с CRM-системой

Основные возможности системы

🎯 Управление визитами

Полный цикл управления визитами медицинских представителей с цветовой индикацией статусов, планированием и отслеживанием результатов.

🏥 Учёт клиник

Детальная база клиник с возможностью синхронизации с Bitrix24, включая динамические поля и историю взаимодействий.

👨‍⚕️ Управление врачами

Ведение базы врачей с привязкой к клиникам, отслеживание контактов и истории визитов.

🔄 Интеграция Bitrix24

Двусторонняя синхронизация всех данных с Bitrix24, включая визиты, клиники, врачей и контакты.

⚙️ Динамические поля

Настраиваемые поля с поддержкой разных типов (текст, дата, время, списки) для гибкой адаптации под бизнес-процессы.

📱 Мобильное приложение

Прогрессивное веб-приложение (PWA) с адаптивным дизайном, работающее на всех устройствах.

API и интеграции

REST API Endpoints

🔐 Аутентификация

  • POST /auth/register
  • POST /auth/login
  • GET /auth/me

📅 Визиты

  • GET /visits
  • POST /visits
  • PUT /visits/{id}
  • DELETE /visits/{id}

🏥 Клиники

  • GET /clinics
  • POST /clinics
  • PUT /clinics/{id}
  • GET /search/inn/{inn}

🔄 Синхронизация

  • POST /bitrix/sync/clinics
  • POST /bitrix/sync/visits
  • POST /bitrix/sync/doctors

Интеграция с Bitrix24

🔗 Особенности интеграции

  • API Endpoint: https://crmwest.ru/rest/156/fnonb6nklg81kzy1/
  • Визиты: Smart Process 1054
  • Клиники: Компании (Companies)
  • Врачи: Контакты с типом DOCTOR
  • Синхронизация: Двусторонняя с отслеживанием статуса
